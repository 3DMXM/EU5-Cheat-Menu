# 欧陆风云5 作弊菜单 / Europa Universalis V Cheat Menu

这是一个为《欧陆风云5》制作的作弊菜单模组,提供了方便的游戏内作弊功能。

This is a cheat menu mod for Europa Universalis V that provides convenient in-game cheat functions.

## 功能特性 / Features

### 经济 / Economy
- 添加 10,000 金币 / Add 10,000 Gold
- 添加 50,000 人力 / Add 50,000 Manpower  
- 添加 50,000 水手 / Add 50,000 Sailors

### 稳定度与威望 / Stability & Prestige
- 最大稳定度 / Max Stability (3)
- 最大正统性 / Max Legitimacy (100)
- 添加 100 威望 / Add 100 Prestige

### 君主点数 / Power Points
- 添加 999 点数 / Add 999 Power Points (Admin/Diplomatic/Military)

### 发展 / Development
- 切换即时建造 / Toggle Instant Construction
- 添加 10 发展度 / Add 10 Development (to selected province)

### 军事 / Military
- 最大陆军职业度 / Max Army Professionalism (100)
- 赢得所有战争 / Win All Wars

### 地图视野 / Map Vision
- 切换战争迷雾 / Toggle Fog of War
- 揭示整个地图 / Reveal Entire Map

### 其他 / Other
- 切换 AI / Toggle AI
- 即时吞并 / Instant Integration
- 切换无条件接受 / Toggle Yes-Man Mode

## 使用方法 / How to Use

1. 启动游戏后,在游戏界面右上角会出现一个"CM"按钮
2. 点击按钮打开作弊菜单
3. 选择你想要的作弊功能

1. After launching the game, a "CM" button will appear in the top-right corner
2. Click the button to open the cheat menu
3. Select the cheat function you want
